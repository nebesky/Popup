Масштабируемое окно
У данного окна размеры заведомо прописаны в CSS, если потребуется изменить ширину и высоту, необходимо прописывать эти стили для каждого отдельно взятого окна.

Самый простой, с точки зрения реализации, вариант. Используется когда на сайте немного всплывающих окон фиксированного размера.
Окно с фиксированным размером

Можно использовать для вывода контента, ширину которого мы можем задать в процентах. Для выравнивания по высоте же потребуется javascript.
Динамическое окно
Фиксированная ширина и динамическая высота.

Позволяет строить окна любого необходимого размера.

Из недостатков - легкое подрагивание при масштабировании из-за пересчета высоты и ширины.
